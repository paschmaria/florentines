== Description ==
Forma is a bold, minimal WordPress theme that empowers you to easily create visually powerful content. 100% responsive, customizable and easy to use.

* Responsive Layout
* Optional featured posts carousel
* Masonry grid or classic posts layout
* Optional hidden sidebar on the right hand-side
* Optional author information box in the single post
* Gallery, video, link and blockquote post formats
* Social links menu in the site footer
* Custom Colors
* Jetpack infinite scroll support
* Contact Form 7 support
* WP Instagram Widget support

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Forma in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
